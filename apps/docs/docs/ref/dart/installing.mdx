### Download & Install the Binary

<RefSubLayout.EducationRow>
  <RefSubLayout.Details>

    Download the binary for the latest Dijets CLI release or build it from source.

  </RefSubLayout.Details>

  <RefSubLayout.Examples>

    <Tabs
      size="small"
      type="underlined"
      defaultActiveId="flutter"
    >
      <TabPanel id="flutter" label="Download the Binary">

        ```bash Terminal
        curl -sSfL https://raw.githubusercontent.com/ava-labs/avalanche-cli/main/scripts/install.sh | sh -s
        ```

      </TabPanel>
      <TabPanel id="dart" label="Build from source">

        ```bash Terminal
        git clone https://github.com/ava-labs/avalanche-cli && cd avalanche-cli && ./scripts/build.sh
        ```

      </TabPanel>
    </Tabs>

  </RefSubLayout.Examples>
</RefSubLayout.EducationRow>
